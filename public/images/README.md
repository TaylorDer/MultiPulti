# Папка для изображений

Помещайте сюда все изображения, которые будут использоваться в пособии.

## Использование

В Markdown контенте (файл `src/data/chapters.ts`) используйте:

```markdown
![Описание изображения](/images/your-image.jpg)
```

## Форматы

Рекомендуемые форматы:
- **JPG/JPEG** - для фотографий
- **PNG** - для графиков, схем с прозрачностью
- **SVG** - для векторной графики
- **GIF** - для анимаций (если необходимо)

## Размеры

Для оптимальной производительности:
- Максимальная ширина: 1200px
- Оптимизируйте изображения перед загрузкой
