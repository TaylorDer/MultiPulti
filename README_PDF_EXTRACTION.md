# Инструкция по извлечению текста из PDF

## Способ 1: Автоматическое извлечение (рекомендуется)

1. **Скопируйте файл** `litovka (2).pdf` в папку проекта

2. **Установите библиотеку для работы с PDF:**
   ```bash
   pip install pdfplumber
   ```
   или
   ```bash
   pip install PyPDF2
   ```

3. **Запустите скрипт:**
   ```bash
   python extract_pdf.py
   ```
   или с указанием пути:
   ```bash
   python extract_pdf.py "C:\путь\к\файлу\litovka (2).pdf"
   ```

4. Текст будет сохранен в файл `extracted_text.txt`

5. Сообщите мне, и я добавлю текст в пособие

## Способ 2: Ручное копирование

1. Откройте PDF файл
2. Выделите весь текст (Ctrl+A)
3. Скопируйте (Ctrl+C)
4. Вставьте текст в чат
5. Я добавлю его в пособие с правильным форматированием

## Способ 3: Указать путь к файлу

Если файл находится в другом месте, укажите полный путь к нему, и я попробую его прочитать.
